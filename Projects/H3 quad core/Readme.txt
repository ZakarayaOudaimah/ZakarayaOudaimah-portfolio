Mini computer 1.2Ghz ->H3 quad core Cortex-A7 processor @ 1.2 GHz with Mali-400MP2 GPU
-System Memory –> 512MbX16 = 1GB of DDR3 RAM full compatible with (H5TC8G63AMR-PBA)
-Storage –> micro SD card slot, and NAND Flash (Fully compatible with eMMC)
-Video Output --> HDMI up to 1080p60
-Connectivity –> 10/100M Ethernet
-USB –> 2x USB host ports + 1x mini USB OTG port
-Expansion –> Optional 2x 40-pin headers with GPIOs, I2C, SPI, etc…
-Debugging –> UART console
-4x user buttons, 4x mounting holes
-Power Supply –> 5V via micro USB OTG port or EXT +5V or +15V
-Dimensions –> 3050×2000 mil (6-layer PCB).

I designed it using Altium Designer 21.1.1.1 with impedance control for 6 stack-up layers.

Note: for DDR3, you can use 1Gb X 4 or 2Gb X 8.
